Rules:

You will have 6 attempts to guess a hexadecimal code for the given color.

After submitting a guess, if a square turns yellow, it means that you have used a correct character, but it is in the wrong position.

If a square turns green, the character is in the correct position.

If a square is grey, the character you have used in not in the code at all.
